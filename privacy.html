Privacy policy

Quota Peek stores monitor data locally and connects directly to provider billing endpoints.

Last updated: 2026-06-26

Overview

Quota Peek is a Windows application published by Amuin. It helps users monitor AI API balances, costs, and usage.

Data used by the app

The app stores user-provided API keys, monitor names, refresh preferences, and locally saved usage snapshots on the user's device.

API keys are stored with Windows protected local storage mechanisms and are not uploaded to Amuin-operated servers by the current app release.

Data not collected by the app

Quota Peek does not collect prompts, model responses, identity documents, phone numbers, photos, audio recordings, contacts, or precise location.

The app does not provide a cloud relay for AI usage queries in the current release.

Network behavior

When a user configures or refreshes a monitor, the app may connect directly from that Windows device to the billing or usage endpoints of the configured provider, such as OpenAI, DeepSeek, or Moonshot.

These network requests are initiated by user configuration or refresh actions.

Retention

Configuration data and stored snapshots remain on the local device until the user deletes them or removes the app.
